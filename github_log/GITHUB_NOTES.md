### Tabintating

Why GitHub blocks this:
GitHub uses a library called HTML Pipeline that sanitizes all input. It filters out target attributes to prevent Tabnabbing, a phishing attack where a newly opened page can change the URL of the original page (GitHub) to a fake login screen without the user noticing.

Summary: If you are viewing this on GitHub.com, there is unfortunately no way to force a new tab with raw HTML or Markdown. The platform intentionally blocks it for security.
